cmc learning quiz

## Compile on Ubuntu/Mint

### Prerequirements

    # apt install texlive make texlive-generic-extra xzdec texlive-lang-cyrillic
    $ tlmgr init-usertree
    $ tlmgr option repository ftp://tug.org/historic/systems/texlive/2015/tlnet-final
    $ tlmgr install a4wide

### Compile

    $ make

    It's easy, right?
